Project Dragon

Project Type: Game Development

Software Used: Visual Studio, Github Desktop

Language(s) Used: C++

Primary Role(s): Solo Project

Using SDL2 and C++, I set out to create what I thought would be a simple game where you play as a dragon and shoot enemies. However, I ended up leveraging a significant amount of C++ to implement various game engine-like features, replicating functionalities I missed from Unity. The process was exhausting at times, but seeing everything come together into a polished system was incredibly rewarding.
