Wormhole

Project Type: Software Development

Software Used: Visual Studio, Github Desktop

Language(s) Used: C#

Primary Role(s): Solo Project

A complex solution to a simple problem. Friends and I wanted to play ARMA 3 together but the setup for that took time and technical proficiency which not everyone in the group had. Leading us to one-by-one walk people through installing and setting up all the things to play together like TeamSpeak, task force radio and ZeroTier.

Since there’s almost 20 of us wanting to play I devised a complex solution to the simple problem. Wormhole! An application that installs and configures software for the user so they can quickly and easily get themselves ready to play with us!

View Source Code

Highlights

- Installs TeamSpeak and Taskforce radio addon.
- Installs ZeroTier and joins the ZeroTier network.
- Utilises WPF technologies for a modern look.
- Utilises ClickOnce rigged to a GitHub repository for automatic updates.
