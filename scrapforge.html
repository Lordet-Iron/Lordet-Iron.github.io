ScrapForge

Project Type: Game Development

Software Used: Unity, Visual Studio

Language(s) Used: C#

Primary Role(s): Head of Code & Engine Developer

Scrap Forge is an Isometric 3D Action-Adventure game project I was apart of as Head of Code during my 2nd year at BCU University. The team spanned over 50+ members about 12 of whom were developers. The game is set on a forgotten planet, collapsed, unloved and filled with junk littered everywhere.

My Contributions

- Designed foundational game architecture.
- Designed and implemented an item creator interface for designers to contribute with.
- Created and managed the gitlab repository, authoring and merging branches across the whole project.
- Collaborated with artists, producers, designers and other developers to bring the project vision to life.
- Created and reviewed code standards. - Attended and gave weekly presentations to the team.
